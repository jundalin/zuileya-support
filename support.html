📱 App使用指南

醉了么是一款专为饮酒场景设计的轻社交工具，集安全守护、娱乐评测、美好记录于一体。所有数据本地处理，保护您的隐私安全。

核心功能：

  • 醉签：酒局倒计时守护，自动报备，超时预警
  • 醉测：娱乐语音评测，创意台词，方言灌口
  • 醉圈：微醺碎忆，记录美好时光
  • 醉伴：一键代驾，安全抵达守护

❓ 常见问题

Q1: 如何使用醉签功能？

A: 打开"醉签"页面 → 设置预计酒局时长（1-12小时）→ 点击"开始醉签" → 确认温馨提醒后，系统会自动向紧急联系人发送酒局守护提醒短信 → 开始倒计时。倒计时结束前5分钟和3分钟会提醒您报备平安，结束后可选择向紧急联系人报备平安短信。

Q2: 醉测功能如何使用？

A: 打开"醉测"页面 → 选择分类（如"绕口令地狱级"、"方言灌口·醉乡音大赏"等）→ 选择一句台词 → 点击"录制基准"先录制正常状态 → 然后点击"开始挑战"录制当前状态 → 系统会对比分析差异，生成娱乐评分和幽默点评。结果仅供娱乐参考，严禁酒后驾驶！

Q3: 醉测功能的评分准确吗？

A: 醉测功能仅供娱乐参考，通过语音对比分析差异，生成幽默评分和创意点评。结果不代表真实酒精含量，不能作为酒后驾驶的判断依据。我们强烈建议：喝酒不开车，开车不喝酒！

Q4: 醉圈-微醺碎忆如何使用？

A: 打开"醉圈"页面 → 可以选择"碎忆时间"查看时间线，或"那年今天"查看历史上的今天 → 点击右上角"+"按钮 → 选择添加照片留念、语音留念或文字留念 → 记录您的美好微醺时刻。所有记录仅保存在您的设备本地，不会上传到任何服务器。

Q5: 醉伴-一键微代驾如何使用？

A: 打开"醉伴"页面 → 在"代驾服务"区域设置车牌号和目的地 → 选择代驾服务商（滴滴代驾或高德代驾）→ App会自动预填起点、目的地和车牌号信息 → 跳转到对应App完成呼叫。请确保已安装对应的代驾App。

Q6: 醉伴-安全抵达如何使用？

A: 打开"醉伴"页面 → 先设置紧急联系人 → 选择预计到家时间 → 点击"开始守护" → 系统会计算您到家的预计时间，并在超时未确认时向紧急联系人发送预警短信（包含实时位置）。到家后请点击"提前到达"完成守护。

Q7: 为什么需要位置权限？

A: 位置权限仅用于"醉伴-安全抵达"功能，用于计算您到家的预计时间，并在超时预警时提供实时位置给紧急联系人。位置数据仅在使用时获取，不会持续追踪，也不会存储到服务器。

Q8: 为什么需要麦克风权限？

A: 麦克风权限用于两个功能：1) "醉测"功能的语音评测；2) "醉圈-微醺碎忆"的语音留念功能。所有录音文件仅保存在您的设备本地，不会上传到任何服务器。

Q9: 语音和照片记录存储在哪里？

A: 所有语音、照片、文字记录仅存储在您的设备本地UserDefaults中，不会上传到任何服务器。卸载App后数据将被清除。我们不会访问、收集或存储这些内容到服务器。

Q10: 紧急联系人信息会泄露吗？

A: 不会。紧急联系人信息仅保存在您的设备本地，仅用于在特定情况下（如醉签开始、超时预警、安全抵达提醒等）发送报备短信。紧急联系人信息不会分享给第三方，也不会上传到服务器。

Q11: 醉测支持哪些方言和台词？

A: 醉测功能包含多个分类：绕口令地狱级、方言灌口·醉乡音大赏、情感台词等。方言灌口支持全国主要方言，包括四川话、粤语、东北话、上海话、闽南语、客家话等30+地区方言。我们会持续更新更多有趣的台词。

Q12: 短信能自动发送吗？

A: 根据iOS系统限制，我们无法完全自动发送短信。App会自动弹出短信编辑界面，内容已预填完整，您需要点击"发送"按钮完成发送。这是iOS的安全机制，防止应用后台自动发送垃圾信息。
